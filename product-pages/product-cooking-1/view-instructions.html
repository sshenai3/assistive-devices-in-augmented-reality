Instructions

Instructions for viewing the object in the 3D viewer:

  • View, zoom and rotate the composition in your desktop browser:
    • Rotate using left-long click + drag the side of the object you want to view./li>
    • Zoom using Ctrl + middle scroll to zoom in and out to view the details of the object.
  • View and rotate the composition on your phone browser by finger sliding or dragging the object.

Instructions for viewing the object in AR through a web browser:

  • Click on the Icon at the top right corner of the AR window screen and a QR code will appear.
  • Point your phone camera at the scan code to see the object window appear.
  • Choose Object mode to see the 3D view of the object using simple gestures like zoom, rotate using the pinch of your fingers.
  • Choose AR mode and follow instructions that appear on the screen.

Instructions for viewing the object in AR through a smartphone/ tablet:

  • Tap on the Icon at the top right corner of the AR window screen and this will take you to another screen to view the object in AR.
  • Choose AR mode and follow instructions that appear on the screen.
  • Choose Object mode to see the 3D view of the object using simple gestures like zoom, rotate using the pinch of your fingers.
